# 💰 Expense Tracker Web App

Aplicación web para registrar y visualizar gastos personales.

Construida con **Python y FastAPI**, permite agregar, eliminar y visualizar gastos con estadísticas y gráficos.

---

## 🚀 Tecnologías utilizadas

- Python
- FastAPI
- SQLite
- TailwindCSS
- Chart.js
- Jinja2

---

## ✨ Funcionalidades

- Agregar gastos
- Eliminar gastos
- Dashboard con estadísticas
- Gráfico de gastos por categoría
- Interfaz moderna con TailwindCSS
- Base de datos SQLite

---

## 📊 Dashboard

La aplicación muestra:

- Total gastado
- Número de gastos
- Categoría principal
- Gráfico de distribución

---

## ⚙️ Cómo ejecutar el proyecto

1. Clonar repositorio

git clone https://github.com/uwushiro/python-expense-tracker.git

2. Entrar en la carpeta

cd python-expense-tracker

3. Instalar dependencias

pip install fastapi uvicorn jinja2

4. Ejecutar servidor

python -m uvicorn api:app --reload

5. Abrir en navegador

http://127.0.0.1:8000
